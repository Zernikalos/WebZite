ZKeyCode

enum ZKeyCode : Enum<ZKeyCode> (source)

Enumeration of keyboard key codes, similar to Unity's KeyCode.

This enum provides a type-safe way to reference keyboard keys. Use this enum to check which key was pressed in keyboard events.
