copyInto

fun copyInto(offset: Int, value: ZAlgebraObject): Int(source)

Copies the values from the specified ZAlgebraObject into the collection starting at the location defined by index.

Parameters

index

the position in the collection where the values should be copied.

value

the ZAlgebraObject containing the values to be copied into the collection.

Throws

Error

if the data type of the ZAlgebraObject does not match the collection's data type.
