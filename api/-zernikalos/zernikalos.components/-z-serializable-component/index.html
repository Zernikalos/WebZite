ZSerializableComponent

abstract class ZSerializableComponent<D : ZComponentData>(data: D) : ZBaseComponent(source)

Represents an abstract component in the Zernikalos Engine that can be serialized, meaning it contains data that can be persisted and restored.

Classes inheriting from this component must provide a data object that contains the serializable information. The component uses the ZSerializableImpl mixin to avoid code duplication.

Type Parameters

D

The type of the data associated with this component. It must inherit from ZComponentData.
