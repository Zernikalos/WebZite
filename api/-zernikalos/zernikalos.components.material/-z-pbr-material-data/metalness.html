metalness

var metalness: Float(source)

Defines the metalness of a material.

Metalness is represented as a factor between 0.0 and 1.0. A value of 0.0 signifies a non-metallic surface, while 1.0 represents a fully metallic surface. The value is clamped to ensure it remains within the valid range.
