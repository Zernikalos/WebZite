emissiveIntensity

var emissiveIntensity: Float(source)

Defines the intensity of the emissive property of a material.

The intensity value determines how strongly the material appears to emit light. It can be greater than 1.0 for High Dynamic Range (HDR) effects, but it cannot be negative. If a negative value is provided, it will automatically be clamped to 0.
