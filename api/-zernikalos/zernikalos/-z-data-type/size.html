size

val size: Int(source)

Represents the size of the data type.

The size is determined based on the format of the data type. The formats have the following sizes:

  • NONE: 0

  • SCALAR: 1

  • VEC2: 2

  • VEC3: 3

  • VEC4: 4

  • MAT2: 4

  • MAT3: 9

  • MAT4: 16

  • TEXTURE: 0

  • QUATERNION: 4
